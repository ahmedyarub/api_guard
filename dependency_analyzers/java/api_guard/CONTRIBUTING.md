## Contributing to API Guard

**API Guard is currently a single-author project.**

To ensure consistent architecture and streamline the roadmap towards the Pro version,
we are **not accepting Pull Requests** from the community at this time.

However, we highly value your feedback!
* **Found a bug?** Please open an Issue.
* **Have a feature request?** Please open a Discussion.
* **Want to build a plugin?** The API Guard MCP Server allows you to extend functionality
  locally without modifying the core source code.

Thank you for understanding and for using API Guard Community Edition!
